# 小票边框线说明文档

本文档列出了小票（receipt）中所有的风格线（包括实线和虚线），包括它们的位置、CSS类名、属性和用途。

## 边框线列表

### 1. 小票信息区域底部实线
- **位置**: `.receipt-info` 底部
- **文件**: `style.css` 约 512 行
- **属性**: `border-bottom: 1px solid #333;`
- **用途**: 分隔小票头部信息（订单通知、开始时间、截稿时间等）和制品列表
- **样式**: 实线，1px，颜色 #333

### 2. 表头底部实线
- **位置**: `.receipt-header` 底部
- **文件**: `style.css` 约 558 行
- **属性**: `border-bottom: 1px solid #ccc;`
- **用途**: 分隔表头（"制品 单价 数量 小计"）和制品内容行
- **样式**: 实线，1px，颜色 #ccc

### 3. 制品小计前虚线
- **位置**: `.receipt-summary` 顶部
- **文件**: `style.css` 约 622 行
- **属性**: `border-top: 1px dotted #ccc;`
- **用途**: 分隔制品列表和汇总区域（制品小计、加价、折扣等）
- **样式**: 虚线，1px，颜色 #ccc

### 4. 总金额顶部实线
- **位置**: `.receipt-total` 顶部
- **文件**: `style.css` 约 646 行
- **属性**: `border-top: 1px solid #333;`
- **用途**: 分隔汇总区域和最终总金额（实付金额）
- **样式**: 实线，1px，颜色 #333（深色，更突出）

## 其他风格线

### 5. 制品与制品间的线
- **已移除**。制品与制品之间的 `receipt-divider` 已取消，仅在赠品信息前、赠品大类之间使用。

### 6. 赠品区使用的 receipt-divider
- **位置**: `.receipt-divider`
- **属性**: `border-top: 1px dotted #e5e5e5; max-width: 50%; margin: 0.25rem auto;`
- **用途**: 赠品信息区块前、赠品大类之间
- **样式**: 短虚线（宽度约 50% 居中），1px，颜色 #e5e5e5（浅灰）

## 边框线统计

- **实线总数**: 3条
  - `.receipt-info` 底部 `1px solid #333`
  - `.receipt-header` 底部 `1px solid #ccc`
  - `.receipt-total` 顶部 `1px solid #333`

- **虚线总数**: 2条
  - `.receipt-summary` 顶部 `1px dotted #ccc`（制品小计前）
  - `.receipt-divider` 赠品区 `1px dotted #e5e5e5`（短、浅）

- **总边框线数**: 5条

## 颜色说明

- **#ccc**: 浅灰色，用于表头底部实线、制品小计前虚线
- **#333**: 深灰色，用于小票信息区域底部、总金额上方的实线
- **#e5e5e5**: 更浅灰，用于赠品区短虚线

## CSS 代码位置参考

```css
/* 位置 1: 小票信息区域底部实线 */
.receipt-info {
    border-bottom: 1px solid #333;
}

/* 位置 2: 表头底部实线 */
.receipt-header {
    border-bottom: 1px solid #ccc;
}

/* 位置 3: 制品小计前虚线 */
.receipt-summary {
    border-top: 1px dotted #ccc;
}

/* 位置 4: 总金额顶部实线 */
.receipt-total {
    border-top: 1px solid #333;
}

/* 位置 5: 赠品区短虚线 */
.receipt-divider {
    border-top: 1px dotted #e5e5e5;
    max-width: 50%;
    margin: 0.25rem auto;
}
```
