第一部分：简易上手

1. 打开应用

用浏览器打开项目里的 index.html，首次使用需联网。

2. 首次设置（建议先做）

  1. 点击导航 「设置」。
  2. 填 美工信息（ID、联系方式、默认工期）。
  3. 在 制品设置 添加常用制品（如吧唧、拍立得）并填价格。
  4. 在 系数设置 检查用途、加急、折扣等（有默认值）。
  5. 在 工艺设置 添加工艺（如烫色、白墨）及每层单价。
  6. 点击 「保存设置」。

3. 做一单报价

  1. 点击导航 「排单」 回到主页面。
  2. 点击 + 打开计算抽屉。
  3. 填 单主 ID、联系方式、开始时间、截稿时间（可选）。
  4. 选 用途、加急 等设置选项。
  5. 在 制品 区添加制品：选类型、数量、工艺等。
  6. 点击 「报价」，小票自动生成并加入排单。

4. 查看与保存小票

  • 小票抽屉会弹出：可点 「存图」 导出图片发给单主。
  • 已自动写入排单；在排单页月历和「当前批次」区可查看。
  • 点击排单卡片可 编辑、小票、备注、撤单、废稿、结单。

5. 常用入口

入口作用
排单主页面，月历与待办
+打开计算抽屉
设置美工信息、制品、系数、工艺
排单页右上 统计查看订单数、收入等
排单页右上 记录历史记录列表

第二部分：按页面详细描述

以下按页面、抽屉、弹窗逐一说明用法，方便按界面查找操作。

三、排单页（主页面）

3.1 如何进入

点击导航 「排单」；或从记录页/统计页点击「返回」。

3.2 区域与操作

工具栏（顶部）

  • ‹ / ›：月历上一月 / 下一月。
  • 日期按钮（显示年月）：点击打开日期选择器，选某日后月历跳到该月。
  • 昼夜模式：点击切换；长按（或右键）跟随系统。
  • 统计 / 记录：进入对应页。

月历区：按开始/截稿时间显示排单彩条；跨天显示跨日彩条。点击某条彩条：弹出该单的操作菜单。

当前批次（todo 区）

  • 筛选：当日（默认）/ 当月 / 待排 / 所有。Today 按钮：选中非今天时出现，点后回今日。
  • 排单列表：每项前有复选框，勾选表示制品已完成（进度跟进）。点击排单模块：弹出操作菜单。

四、计算抽屉

4.1 如何打开

点击导航 +：新建。排单卡片「编辑」：加载该单，改完可覆盖或另存。

4.2 如何关闭

左上 ×：不保存。报价：生成小票并加入排单，自动关闭并打开小票。另存为新单（仅编辑模式）：保存为新排单，关闭，不打开小票。

4.3 填写内容

区块内容
单主信息单主 ID、联系方式、下单/开始/截稿时间；「备注」打开备注弹窗
设置选项用途、加急、同模、折扣、平台费；右侧「模版填表」加载模板
其他费用抠图费等，预设或自定义
制品/赠品类型、单双面、数量、同模、工艺

4.4 底部按钮

报价：生成并加入（编辑模式下覆盖原单）。另存为新单：仅编辑模式显示。保存为模板：输入名称保存当前组合。

五、小票抽屉

5.1 如何打开

计算抽屉点「报价」后自动打开；或排单卡片「小票」。

5.2 顶部操作

排单：保存到排单列表。主题：下拉选主题。小票设置：打开设置面板。存图：导出图片。

六、小票设置面板（底部弹出）

小票抽屉内点「小票设置」打开。三个标签：设置（头图/尾图/标题等）、主题（边框/底纹）、字体。底部：清除设置、保存设置。

七、排单卡片操作弹窗

在月历上点击某条排单彩条，或在下方 todo 区点击某个排单模块，会弹出操作菜单，包含：

操作说明
编辑关闭菜单并打开计算抽屉，将该单数据加载到计算页。改完后可二选一：点击 「报价」 覆盖原记录；或点击 「另存为新单」 保存为一条新排单（原单不变）。
小票打开该单的小票视图，可查看报价明细；在此可保存图片或再点「排单」写回历史。
备注打开备注弹窗，可填写或查看该单的备注（仅本地保存，不会打在小票上）。
撤单打开结算弹窗，选择撤单（跑单），按设置页「跑单相关」规则填写跑单费金额等，确认后该单状态变为「已撤单」。
废稿打开结算弹窗，选择废稿，按设置页「废稿相关」规则填写废稿费金额等，确认后该单状态变为「有废稿」。
结单打开结算弹窗，选择正常结单，填写本次收款（默认本次应收）、优惠原因（每项单行：名称、减去/折扣、数值），会显示新平台费与实际单主支付，确认后该单状态变为「已结单」。

结算弹窗中的金额、备注会保存到该条记录，统计页会按结算结果统计收入/撤单/废稿金额。

八、结算弹窗

排单卡片选「撤单」「废稿」「结单」后打开。第一步选类型；第二步按类型填写（撤单：退全款或收跑单费；废稿：按设置页规则填；结单：填本次收款，默认本次应收，可勾选优惠原因、显示新平台费与实际单主支付）；第三步点「确定」保存。

九、记录页

排单页右上点「记录」进入；左上「返回」退出。

顶部：导入（选 JSON 后弹出「导入方式」弹窗：覆盖/合并）、导出（弹层：JSON/Excel）。

搜索/排序/筛选：搜索框、排序图标（弹层选时间/价格/单主）、筛选图标（打开筛选抽屉）。

筛选抽屉：时间范围、状态、价格范围；全选、批量删除。点「应用」关闭。

导入方式弹窗：记录页点「导入」并选 JSON 后出现；选「覆盖本机数据」或「新增合并记录」。

历史记录数量不限。数据在本地，换设备或清空会丢失，请及时导出备份。

十、统计页

排单页右上点「统计」进入；左上「返回」退出。

快捷时间：本月 / 本年 / 全部 / 自定义。

内容：KPI 卡片、环比/同比、趋势图、Top 制品榜。顶部：导出 Excel、筛选。

筛选抽屉：时间范围、金额口径、赠品计入方式、订单状态。点「应用」关闭。

十一、设置页

导航「设置」进入。美工信息、制品设置、系数设置、结算配置、工艺设置。

添加制品弹窗：制品设置→「添加」。填分类、名称、计价方式及价格。

添加系数弹窗：系数设置→「添加」。填大类、小类、系数值。

添加工艺弹窗：工艺设置→「添加」。填名称、每层单价。

修改后点 「保存设置」；可 「恢复默认设置」。

十二、订单备注弹窗

计算页单主「备注」或排单卡片「备注」打开。查看模式只读；编辑模式可输入，智能读取、复制。点 × 或「确定」关闭。

十三、模板的保存与使用

保存：计算抽屉底部「保存为模板」，输入名称。加载：「设置选项」右侧「模版填表」→ 选模板 → 加载。模版填表内可删除模板。

十四、价格公式（供参考）

  • 单件制品价 = 基础价 × (1 + 同模系数×(柄图数−1)) + 工艺费用×工艺层数
  • 总价 = 各制品价相加 → 乘上加价类系数、折扣类系数 → 再加上其他费用合计与平台手续费

具体系数、工艺单价、其他费用以 设置页 当前配置为准。

十五、注意事项

  1. 数据在本地：设置与历史都存在本机浏览器，清空缓存或换设备会丢失，重要内容请保存图片或导出 JSON/Excel。
  2. 跨端同步：记录页导入/导出 JSON，可选择覆盖或合并，用于不同设备间手动同步。
  3. 历史记录：数量不做限制，建议定期导出备份。
  4. 打印/导出：若小票被裁切，可调整打印边距或导出时缩放。
  5. 移动端：筛选采用抽屉式，点击「筛选」按钮打开。
