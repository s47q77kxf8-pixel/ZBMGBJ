# MG报价工具 使用说明

## 一、概述
MG报价工具是一款专门用于制作各类制品（如吧唧、纸片、立牌等）报价单的在线应用，支持多种计价方式和灵活的配置选项。

## 二、界面导航
应用分为三个主要页面：
- **计算页**：输入客户信息、设置参数、添加制品和赠品
- **报价页**：查看生成的报价单、保存历史记录
- **设置页**：配置制品类型、工艺、系数等参数

## 三、计算页功能

### 3.1 单主信息
- **单主ID**：输入客户标识号
- **联系方式**：选择联系方式类型（QQ/微信/米画师）并输入具体信息
- **开始时间**：可选填写项目开始日期
- **截稿时间**：设置交付截止日期

### 3.2 设置选项
- **用途**：选择项目用途（自用/无盈利/同人商用、不公开展示、买断、企业/书店/出版社等）
- **加急**：选择加急程度（无、一周、72H、48H、24H）
- **同模系数**：选择同模制作比例（改字色柄图、改字色柄图元素）
- **折扣系数**：选择折扣（无、上次合作寄样）
- **平台手续费**：选择平台费用（无、米画师、画加）

### 3.3 其他费用
- 从预设类别选择或自定义费用名称和金额
- 点击"+ 其他费用"添加更多费用项

### 3.4 制品和赠品管理
- **添加制品/赠品**：点击"+ 制品"或"+ 赠品"按钮
- **制品类型**：从下拉菜单选择制品类型
- **单双面选择**：对支持单双面的制品，选择单面或双面
- **数量**：设置制品/赠品数量
- **同模**：勾选是否同模制作
- **工艺设置**：为制品或赠品添加各种工艺（烫色、白墨、UV等）

### 3.5 生成报价
- 点击"计算报价"按钮生成报价单

## 四、报价页功能

### 4.1 报价单查看
- 显示完整的报价单，包含制品明细、费用汇总、最终总价等

### 4.2 功能按钮
- **小票设置**：自定义报价单头部和尾部内容
- **保存图片**：将报价单保存为图片
- **保存历史**：将当前报价保存到历史记录

### 4.3 小票设置
- **头部图片**：上传自定义头部图片
- **小票标题**：设置标题文字
- **订单通知**：设置订单通知信息
- **显示选项**：控制开始时间、截稿时间、设计师、联系方式的显示
- **自定义文本**：添加自定义文本
- **尾部设置**：设置尾部文本和图片

### 4.4 历史记录
- 在报价页下方显示历史记录
- 可通过关键词搜索历史记录
- 可查看、加载或删除历史记录

## 五、设置页功能

### 5.1 美工信息
- 设置美工ID、联系方式和默认工期

### 5.2 制品设置
- **添加制品设置**：点击"+ 制品设置"按钮
- **计价方式**：
  - 固定价：单一固定价格
  - 单双面价：单面和双面分别定价
  - 基础+递增价：基础配置+额外配置价格
- **分类管理**：可按分类组织制品

### 5.3 系数设置
- **用途系数**：管理不同用途的系数
- **加急系数**：管理不同加急程度的系数
- **同模系数**：管理同模制作的系数
- **折扣系数**：管理折扣选项
- **平台手续费**：管理平台费用

### 5.4 工艺设置
- 添加和管理各种工艺类型及价格
- 每种工艺可设置每层价格

### 5.5 其他费用
- 管理其他费用类别

### 5.6 保存设置
- 点击"保存设置"保存更改
- 可通过"恢复默认设置"重置为默认值

## 六、计价方式说明

### 6.1 固定价
- 制品有固定价格，不随其他因素变化

### 6.2 单双面价
- **单面**：显示为"基础配置(单面)"
- **双面**：显示为"基础配置(双面)"
- 单面和双面价格不同

### 6.3 基础+递增价
- **基础配置**：显示为"基础配置(具体配置)"
- **递增配置**：显示为"+数量单位 名称"格式

## 七、操作流程

### 7.1 创建报价
1. 在计算页填写单主信息
2. 设置各项系数参数
3. 添加制品和赠品
4. 选择需要的工艺
5. 点击"计算报价"生成报价单

### 7.2 保存和分享
1. 生成报价后点击"保存历史"保存记录
2. 点击"保存图片"保存报价单图片
3. 可在报价页查看历史记录

### 7.3 自定义设置
1. 在设置页配置所需的制品类型、工艺和系数
2. 点击"保存设置"保存更改

## 八、注意事项

1. **数据存储**：所有数据存储在浏览器本地，清除浏览器数据会丢失历史记录
2. **兼容性**：建议使用现代浏览器（Chrome、Firefox、Edge等）
3. **打印设置**：打印时建议调整为无边距或最小边距
4. **系数选择**：根据实际需求合理选择各项系数

## 九、移动端使用
- 网页支持响应式布局，可在手机上正常操作
- 底部导航栏便于在不同页面间切换

这份使用说明基于实际的网页界面和功能，更准确地反映了MG报价工具的使用方法。
