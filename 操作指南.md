## 第一部分：简易上手

### 1. 打开应用

用浏览器打开项目里的 **`index.html`**，首次使用需联网。

### 2. 首次设置（建议先做）

1. 点击导航 **「设置」**。
2. 填 **美工信息**（ID、联系方式、默认工期）。
3. 在 **制品设置** 添加常用制品（如吧唧、拍立得）并填价格。
4. 在 **系数设置** 检查用途、加急、折扣等（有默认值）。
5. 在 **工艺设置** 添加工艺（如烫色、白墨）及每层单价。
6. 点击 **「保存设置」**。

### 3. 做一单报价

1. 点击导航 **「排单」** 回到主页面。
2. 点击 **+** 打开计算抽屉。
3. 填 **单主 ID**、**联系方式**、**开始时间**、**截稿时间**（可选）。
4. 选 **用途**、**加急** 等设置选项。
5. 在 **制品** 区添加制品：选类型、数量、工艺等。
6. 点击 **「报价」**，小票自动生成并加入排单。

### 4. 查看与保存小票

- 小票抽屉会弹出：可点 **「存图」** 导出图片发给单主。
- 已自动写入排单；在排单页月历和「当前批次」区可查看。
- 点击排单卡片可 **编辑**、**小票**、**备注**、**撤单**、**废稿**、**结单**。

### 5. 常用入口

| 入口 | 作用 |
|------|------|
| **排单** | 主页面，月历与待办 |
| **+** | 打开计算抽屉 |
| **设置** | 美工信息、制品、系数、工艺 |
| 排单页右上 **统计** | 查看订单数、收入等 |
| 排单页右上 **记录** | 历史记录列表 |

## 三、排单页（主页面）

### 3.1 如何进入
点击导航 **「排单」**；或从记录页/统计页点击「返回」。

### 3.2 区域与操作

**工具栏（顶部）**
- **‹** / **›**：月历上一月 / 下一月。
- **日期按钮**（显示年月）：点击打开日期选择器，选某日后月历跳到该月。
- **昼夜模式**：点击切换；长按（或右键）跟随系统。
- **统计**：进入统计页。
- **记录**：进入记录页。

**月历区**
- 显示当月日历，按**开始时间**和**截稿时间**在对应日期显示排单彩条；跨天显示跨日彩条。
- **点击某条彩条**：弹出该单的操作菜单（见第五节）。

**当前批次（todo 区）**
- **筛选按钮**：**当日**（默认）/ **当月** / **待排** / **所有**。选「待排」只显示未填开始/截稿的；选「所有」显示全部未结单/未撤单的。
- **Today 按钮**：当选中日期非今天时出现，点击回到今日。
- **排单列表**：每个排单以模块展示，下列制品项；每项前有**复选框**，勾选表示该制品已完成（用于进度跟进）。**点击排单模块**：弹出操作菜单。
- **进度勾选**：制品完成勾选状态会保存，便于进度管理。

## 四、计算抽屉

### 4.1 如何打开
- 点击导航 **+**：新建报价。
- 排单卡片操作中选 **「编辑」**：加载该单到计算页，改完可覆盖或另存。

### 4.2 如何关闭
- 点击左上角 **×**：关闭，不保存。
- 点击 **「报价」**：生成小票并加入排单，关闭抽屉并打开小票抽屉。
- 编辑模式下点击 **「另存为新单」**：保存为新排单，关闭抽屉，不打开小票。

### 4.3 填写内容

| 区块 | 内容 | 说明 |
|------|------|------|
| 单主信息 | 单主 ID、联系方式、下单/开始/截稿时间 | 点击「备注」可打开备注弹窗；填开始时间+工期可自动算截稿 |
| 设置选项 | 用途、加急、同模、折扣、平台费 | 右侧 **「模版填表」** 可加载已保存模板 |
| 其他费用 | 抠图费等 | 从预设选或自定义 |
| 制品 | 类型、单双面、数量、同模、工艺 | 可多行 |
| 赠品 | 同上 | 小票标为「赠品」可显示划线价 |

### 4.4 底部按钮
- **报价**：生成小票并加入排单（编辑模式下覆盖原单）。
- **另存为新单**：仅编辑模式显示；保存为新排单，原单不改。
- **保存为模板**：将当前制品+赠品+设置选项保存为模板，输入名称即可。

### 4.5 模版填表（下拉）
- 点击「模版填表」展开下拉：选模板 → 点「加载」填入；可删除模板。

<a id="s5"></a>
## 五、小票抽屉

### 5.1 如何打开
- 计算抽屉内点击 **「报价」** 后自动打开。
- 排单卡片操作中选 **「小票」**：查看该单的小票。

### 5.2 如何关闭
- 点击左上角 **×** 或点击遮罩。

### 5.3 顶部操作
- **排单**：将当前小票保存到排单列表（若尚未保存）。
- **主题**：下拉选小票主题（经典/现代/深色等）。
- **小票设置**：打开小票设置面板（头图/尾图/标题/主题/字体）。
- **存图**：将当前小票导出为图片。

### 5.4 约定实收（若显示）
- 可填写与单主约定的金额；**向下** / **取整到十** / **取整到百** 可快速取整。

<a id="s6"></a>
## 六、小票设置面板（底部弹出）

### 6.1 如何打开
小票抽屉内点击 **「小票设置」** 图标。

### 6.2 如何关闭
- 点击 **「保存设置」** 或遮罩。

### 6.3 三个标签页

| 标签 | 内容 |
|------|------|
| **设置** | 头图、尾图、标题、标题后文本、订单说明；是否显示时间、设计师信息、定金；是否付定金 |
| **主题** | 预设主题、自定义边框颜色、底纹样式 |
| **字体** | 统一字体或分类（标题/正文/数字/汇总）；自定义字体导入（.ttf/.otf） |

### 6.4 底部按钮
- **清除设置**：重置小票自定义。
- **保存设置**：应用并关闭。

<a id="s7"></a>
## 七、排单卡片操作弹窗

### 7.1 如何打开
- 月历上点击某条排单彩条。
- 当前批次（todo 区）点击某个排单模块。

### 7.2 如何关闭
- 点击遮罩或右上角 **×**。

### 7.3 六个操作

| 操作 | 说明 |
|------|------|
| **编辑** | 关闭弹窗并打开计算抽屉，加载该单；改完后「报价」覆盖或「另存为新单」 |
| **小票** | 打开小票抽屉，查看该单报价；可存图或点「排单」写回历史 |
| **备注** | 打开订单备注弹窗，填写或查看备注（仅本地，不打在小票上） |
| **撤单** | 打开结算弹窗，选撤单（退全款或收跑单费），填金额等，确认后状态为「已撤单」 |
| **废稿** | 打开结算弹窗，选废稿，按设置页废稿费规则填写，确认后状态为「有废稿」 |
| **结单** | 打开结算弹窗，选结单，填实收金额等，确认后状态为「已结单」 |

<a id="s8"></a>
## 八、结算弹窗

### 8.1 如何打开
排单卡片操作中选 **「撤单」** / **「废稿」** / **「结单」**。

### 8.2 如何关闭
- 点击遮罩、右上 **×** 或底部 **「取消」**。

### 8.3 步骤说明

**第一步：选择结算类型**
- **撤单**：退全款或收跑单费。
- **废稿**：收废稿费（规则来自设置页）。
- **结单**：正常结算。

**第二步：按类型填写**
- **撤单**：选「退全款」或「收跑单费」；若收跑单费，选按比例或固定金额，填比例/金额、实收、备注。
- **废稿**：按设置页废稿费模式显示表单；填制品明细、系数、其他费用等，确认废稿费金额。
- **结单**：填实收金额、备注。

**第三步**：点击 **「确定」** 保存，弹窗关闭，该单状态更新。

<a id="s9"></a>
## 九、记录页

### 9.1 如何进入
排单页右上角点击 **「记录」** 图标。

### 9.2 如何返回
点击左上角 **「返回」** 回到排单页。

### 9.3 顶部操作
- **导入**：选择 JSON 文件后，弹出「导入方式选择」弹窗（覆盖/合并）。
- **导出**：点击后弹出选项，选「导出 JSON（同步）」或「导出 Excel」。

### 9.4 搜索、排序、筛选
- **搜索框**：输入关键词筛选；右侧 **×** 清空。
- **排序**：点击排序图标，选时间/价格/单主 ID 升序或降序。
- **筛选**：点击筛选图标，打开筛选抽屉（见下节）。

### 9.5 记录列表
- 每条显示单主 ID、金额、状态；可点击查看详情、编辑、删除等。
- **批量删除**：在筛选抽屉内勾选后，点「批量删除」。

### 9.6 记录页筛选抽屉
- **打开**：点击「筛选」图标。
- **关闭**：点击遮罩或「应用」。
- **内容**：时间范围、状态（待排单/未开始/进行中/已完成/已逾期/已结单/已撤单/有废稿）、价格范围；全选、批量删除。

### 9.7 导入方式选择弹窗
- **何时出现**：记录页点击「导入」并选择 JSON 文件后。
- **选项**：**覆盖本机数据**（用文件替换本机）、**新增合并记录**（仅合并新记录，本机设置不变）。
- 选择后执行导入，弹窗自动关闭。

## 十、统计页

### 10.1 如何进入
排单页右上角点击 **「统计」** 图标。

### 10.2 如何返回
点击左上角 **「返回」** 回到排单页。

### 10.3 快捷时间
- **本月** / **本年** / **全部** / **自定义**：切换统计范围。

### 10.4 顶部操作
- **导出 Excel**：按当前筛选导出。
- **筛选**：打开筛选抽屉。

### 10.5 内容区域
- **KPI 卡片**：订单数、收入、客单价、制品件数、完成率、结单率、撤单/废稿金额等。
- **环比/同比**：与上一周期对比。
- **趋势图**：日/周/月趋势。
- **Top 制品榜**：按金额排序的制品排行。

### 10.6 统计页筛选抽屉
- **打开**：点击「筛选」图标。
- **关闭**：点击遮罩或「应用」。
- **内容**：时间范围、金额口径（实付/制品价）、赠品计入方式、订单状态。

## 十一、设置页

### 11.1 如何进入
点击导航 **「设置」**。

### 11.2 美工信息
- 美工 ID、联系方式、默认工期、定金比例。
- 右上角 **「操作指南」**：打开本指南网页版。

### 11.3 制品设置
- 按分类管理制品，每种选计价方式（固定价/单双面价/基础+递增价）并填价格。
- **添加**：打开添加制品弹窗。
- **导入** / **导出**：Excel 批量导入/导出制品。

### 11.4 系数设置
- 加价类、折扣类、其他类（同模、平台费、背景费、其他费用）。
- **添加**：打开添加系数弹窗。
- **导入** / **导出**：Excel 批量导入/导出系数。

### 11.5 结算配置
- **跑单相关**：撤单费规则（按比例/固定金额）、默认比例/金额。
- **废稿相关**：废稿费模式（按总额比例/按件/固定等）、保底/封顶。

### 11.6 工艺设置
- 工艺名称与每层单价。
- **添加**：打开添加工艺弹窗。

### 11.7 底部按钮
- **保存设置**：保存并生效。
- **恢复默认设置**：还原为默认配置。

### 11.8 添加制品弹窗
- **打开**：制品设置 →「添加」。
- **填写**：分类、制品名称、计价方式及对应价格/配置。
- **保存** / **取消**。

### 11.9 添加系数弹窗
- **打开**：系数设置 →「添加」。
- **填写**：系数大类（加价类/折扣类）、小类、系数值项。
- **保存** / **取消**。

### 11.10 添加工艺弹窗
- **打开**：工艺设置 →「添加」。
- **填写**：工艺名称、价格（每层）。
- **保存** / **取消**。

## 十二、订单备注弹窗

### 12.1 如何打开
- 计算抽屉单主信息处点击「备注」图标。
- 排单卡片操作中选「备注」。

### 12.2 两种模式
- **查看**：只读显示已有备注。
- **编辑**：可输入、修改备注；**智能读取** 可解析格式（如「立绘 单面 2个」）并填入制品；**复制** 复制到剪贴板。

### 12.3 如何关闭
- 点击 **×** 或 **「确定」**。

<a id="s13"></a>
## 十三、模板的保存与使用

- **保存模板**：计算抽屉填好内容后，底部点「保存为模板」，输入名称。
- **加载模板**：计算抽屉「设置选项」右侧「模版填表」→ 选模板 → 点「加载」。
- **删除模板**：模版填表下拉内可删除。

<a id="s14"></a>
## 十四、价格公式（供参考）

- **单件制品价** = 基础价 × (1 + 同模系数×(柄图数−1)) + 工艺费用×工艺层数
- **总价** = 各制品价相加 → 乘上加价类系数、折扣类系数 → 再加上其他费用合计与平台手续费

<a id="s15"></a>
## 十五、注意事项

1. **数据在本地**：设置与历史存于浏览器 localStorage，清空或换设备会丢失，请及时保存图片或导出 JSON/Excel 备份。
2. **跨端同步**：记录页导入/导出 JSON，可选覆盖或合并，用于设备间手动同步。
3. **历史记录**：数量不限，建议定期导出备份。
4. **打印/导出**：小票被裁切时可调整边距或缩放。
5. **移动端**：筛选为抽屉式，点击「筛选」打开。

更多技术说明见 **README.md**。

> 💡 应用内可在**设置页**→「美工信息」标题栏右侧点击「操作指南」（图标+文字）打开本指南网页版。
