## 第一部分：简易上手

### 1. 打开应用

用浏览器打开项目里的 **`index.html`**，首次使用需联网。

### 2. 首次设置（建议先做）

1. 点击导航 **「我的」**。
2. 填 **美工信息**（ID、联系方式、默认工期）。
3. 在 **制品设置** 添加常用制品（如吧唧、拍立得）并填价格。
4. 在 **系数设置** 检查用途、加急、折扣等（有默认值）。
5. 在 **工艺设置** 添加工艺（如烫色、白墨）及每层单价。
6. 修改会**自动保存**到本机；可点底部 **「立即保存」** 再次写入。

### 3. 做一单报价

1. 点击导航 **「排单」** 回到主页面。
2. 点击 **+** 打开计算抽屉。
3. 填 **单主 ID**、**联系方式**、**下单时间**、**开始时间**、**截稿时间**（可选）。
4. 选 **用途**、**加急** 等设置选项。
5. 在 **制品** 区添加制品：选类型、数量、工艺等。
6. 点击 **「小票」**，小票自动生成并加入排单。

### 4. 查看与保存小票

- 小票抽屉会弹出：可点 **「存图」** 导出图片发给单主。
- 已自动写入排单；在排单页月历和「当前批次」区可查看。
- 点击排单卡片可 **编辑**、**小票**、**备注**、**撤单**、**废稿**、**结单**。

### 5. 常用入口

| 入口 | 作用 |
|------|------|
| **排单** | 主页面，月历与待办 |
| **+** | 打开计算抽屉 |
| **我的** | 美工信息、制品、系数、工艺 |
| 排单页右上 **统计** | 查看企划数、收入等 |
| 排单页右上 **记录** | 历史记录列表 |
| **登录** | 打开登录页面，进行账户管理 |
| **客户端页面** | 打开客户需求填写表单 |

## 三、排单页（主页面）

### 3.1 如何进入
点击导航 **「排单」**；或从记录页/统计页点击「返回」。

### 3.2 区域与操作

**工具栏（顶部）**
- **‹** / **›**：月历上一月 / 下一月。
- **日期按钮**（显示年月）：点击打开日期选择器，选某日后月历跳到该月。
- **Today 按钮**：当选中日期非今天时出现，点击回到今日。
- **展开/收起全部彩条**：点击图标按钮可展开或收起所有排单彩条。
- **昼夜模式**：点击切换；长按（或右键）跟随系统。
- **统计**：进入统计页。
- **记录**：进入记录页。

**月历区**
- 显示当月日历，按**开始时间**和**截稿时间**在对应日期显示排单彩条；跨天显示跨日彩条。
- **点击某条彩条**：弹出该单的操作菜单（见第五节）。
- **手势操作（移动端/触控板）**：
  - 双指缩放：缩放月历视图（便于查看密集排单）。

**当前批次（todo 区）**
- **筛选按钮**：**所有** / **当月** / **待排** / **当日**（默认）。选「待排」只显示未填开始/截稿的；选「所有」显示全部未结单/未撤单的。
- **换行按钮**：点击可切换 todo 列表的显示模式（换行/不换行）。
- **排单列表**：每个排单以模块展示，下方列出制品项；每项前有**复选框**，勾选表示该制品已完成（用于进度跟进）。
  - **横向滚动**：制品项过多时支持横向滑动查看。
  - 点击**复选框/制品名**：仅切换完成状态，不弹出菜单。完成项文字会增加划线效果。
  - 点击卡片**空白区域**（包括制品行间的间隙）：弹出该单的操作菜单。
- **进度勾选**：制品完成勾选状态会保存，已结单的排单会自动从 todo 列表中隐藏，月历彩条划线沉底。

## 四、计算抽屉

### 4.1 如何打开
- 点击导航 **+**：新建报价。
- 排单卡片操作中选 **「编辑」**：加载该单到计算页，改完可覆盖或另存。

### 4.2 如何关闭
- 点击左上角 **×**：关闭，不保存。
- 点击 **「小票」**：生成小票并加入排单，关闭抽屉并打开小票抽屉。
- 编辑模式下点击 **「覆盖」**：直接保存到原企划，关闭抽屉，不打开小票。
- 编辑模式下点击 **「新单」**：另存为新排单，原单不改，关闭抽屉，不打开小票。

### 4.3 填写内容

| 区块 | 内容 | 说明 |
|------|------|------|
| 单主信息 | 单主 ID、联系方式、下单/开始/截稿时间 | 点击「备注（智能提取）」可打开备注弹窗，粘贴单主备注后点「智能提取」可自动识别制品与赠品；填开始时间+工期可自动算截稿 |
| 档期占位 | 预计制品数 | 可设置档期占位，方便预留时间 |
| 设置选项 | 用途、加急、同模、折扣、平台费 | 右侧 **「模版填表」** 可加载已保存模板 |
| 每制品增加 | 其他费用 | 可多选，为每个制品添加额外费用 |
| 平台费与定金 | 平台手续费、是否付定金 | 选择平台后自动匹配手续费；可设置是否需要定金 |
| 其他费用 | 抠图费等 | 从预设选或自定义 |
| 制品 | 类型、单双面、数量、同模、工艺 | 可多行 |
| 赠品 | 同上 | 小票标为「赠品」可显示划线价 |

### 4.4 底部按钮
- **小票**：生成小票并加入排单，关闭抽屉并打开小票抽屉。
- **覆盖**：仅编辑模式显示；直接保存到原企划，关闭抽屉，不打开小票。
- **新单**：仅编辑模式显示；另存为新排单，原单不改，关闭抽屉，不打开小票。
- **保存为模板**：将当前制品+赠品+设置选项保存为模板，输入名称即可。

### 4.5 模版填表（下拉）
- 点击「模版填表」展开下拉：选模板 → 点「加载」填入；可删除模板。

## 五、小票抽屉

### 5.1 如何打开
- 计算抽屉内点击 **「小票」** 后自动打开。
- 排单卡片操作中选 **「小票」**：查看该单的小票。

### 5.2 如何关闭
- 点击左上角 **×** 或点击遮罩。

### 5.3 顶部操作
- **保存**：将当前小票保存到排单列表（若尚未保存）。
- **主题**：下拉选小票主题（经典/现代/深色/复古/温馨/清新/粉樱/冰蓝等）。
- **小票设置**：打开小票设置面板（头图/尾图/标题/主题/字体）。
- **复制**：将小票复制为图片。
- **存图**：将当前小票导出为图片。

### 5.4 报价金额 / 需付定金（顶部预览）
- 顶部会展示 **报价金额（不含平台费）**。
- 若在计算页选择「需要定金」，顶部会在报价金额下方展示 **需付定金**（按定金比例实时计算，样式与报价金额一致）。

### 5.5 约定实收
- 可填写与单主约定的金额；**向下** / **取整到十** / **取整到百** 可快速取整。
- 修改约定实收后，需付定金会同步更新。

### 5.6 已收定金
- 点击 **「确认已收定金」** 按钮后，可输入已收定金金额
- 支持快速取整操作
- 可点击「取消」按钮取消已收定金的确认

### 7.2 如何关闭

## 八、结算弹窗

### 8.1 如何打开
排单卡片操作中选 **「撤单」** / **「废稿」** / **「结单」**。

### 8.2 如何关闭
- 点击遮罩、右上 **×** 或底部 **「取消」**。

### 8.3 步骤说明

**第一步：选择结算类型**
- **撤单**：退全款或收跑单费。
- **废稿**：收废稿费（规则来自设置页）。
- **结单**：正常结算。

**第二步：按类型填写**
- **撤单**：选「退全款」或「收跑单费」；若收跑单费，选按比例或固定金额，填比例/金额、实收、备注。
- **废稿**：按设置页废稿费模式显示表单；填制品明细、系数、其他费用等，确认废稿费金额。
- **结单**：勾选或添加**优惠原因**（每项一行：名称、减去/折扣切换、数值）；**本次收款**默认本次应收，可修改；会显示**新平台费**与**实际单主支付**（合计实收 + 新平台费）；填备注后点「确定」。

**第三步**：点击 **「确定」** 保存，弹窗关闭，该单状态更新。

## 九、记录页

### 9.1 如何进入
排单页右上点击 **「记录」** 图标。

### 9.2 如何返回
点击左上角 **「返回」** 回到排单页。

### 9.3 顶部操作
- 无导入/导出功能，相关功能已移至「我的页面」。

### 9.4 搜索、排序、筛选
- **搜索框**：输入关键词筛选；右侧 **×** 清空。
- **排序**：点击排序图标，选时间/价格/单主 ID 升序或降序。
- **筛选**：点击筛选图标，打开筛选抽屉（见下节）。

### 9.5 记录列表
- 每条显示单主 ID、金额、状态；可点击查看详情、编辑、删除等。
- **批量删除**：
  - 在记录列表中勾选要删除的记录
  - 点击「批量删除」按钮
  - 确认删除后，选中的记录将被删除
  - 支持云端同步删除
  - 批量删除按钮会显示选中的记录数量，如「批量删除(3)」

### 9.6 记录页筛选抽屉
- **打开**：点击「筛选」图标。
- **关闭**：点击遮罩或「应用」。
- **内容**：**显示方式**（按月分组 / 全部排序）、时间范围、状态（待排单/未开始/进行中/已完成/已逾期/已结单/已撤单/有废稿）、价格范围；全选、批量删除。选「全部排序」时按当前排序规则整列表显示，不按月份分组。

### 9.7 导入方式选择弹窗
- **何时出现**：记录页点击「导入」并选择 JSON 文件后。
- **选项**：**覆盖本机数据**（用文件替换本机）、**新增合并记录**（仅合并新记录，本机设置不变）。
- 选择后执行导入，弹窗自动关闭。

## 十、统计页

### 10.1 如何进入
排单页右上点击 **「统计」** 图标。

### 10.2 如何返回
点击左上角 **「返回」** 回到排单页。

### 10.3 时间范围（在面板）
- **本月** / **本年** / **全部** / **自定义**：在统计页面板切换；选「本月」或「本年」时可切换月份或年份；选择本月/本年时显示**环比上月/去年**。

### 10.4 顶部操作
- **时光报告**：生成时光报告。
- **筛选**：打开筛选抽屉。

### 10.5 内容区域
- **KPI 卡片**：企划数、收入、客单价、制品件数、完成率、结单率、撤单/废稿金额等。无数据时显示「当前筛选条件下暂无企划数据」。
- **环比/同比**：选本月/本年时与上一周期对比。
- **趋势图**：日/周/月趋势。
- **数据分布**：按用途、加急、折扣等显示单数/金额。
- **制品小类汇总**：横向可滚动查看。
- **Top 单主**：默认显示 10 条，点「更多」展开；**Top 制品榜**：按金额排序的制品排行。
- **查看企划**：点击各项数据旁边的「查看企划」按钮或 KPI 卡片，可跳转到记录页并筛选显示对应的企划

## 二十、系数计算模式

### 20.1 功能说明

系统支持加价类和折扣类系数分别选择计算模式：乘法模式或加法模式。计算逻辑为先算加价类，再算折扣类。

### 20.2 配置位置

在 `script.js` 的 `defaultSettings.pricingCalculationMode` 中配置：

```javascript
pricingCalculationMode: {
    up: 'multiplicative',   // 加价类：'multiplicative'（乘法）或 'additive'（加法）
    down: 'multiplicative'  // 折扣类：'multiplicative'（乘法）或 'additive'（加法）
}
```

### 20.3 计算逻辑

#### 加价类系数

| 模式 | 计算方式 | 示例（用途=2，加急=1.5） |
|------|----------|-------------------------|
| **乘法模式** | 系数相乘 | `2 × 1.5 = 3` → 加价后价格 = 原价 × 3 |
| **加法模式** | 系数增量相加 | `1 + (2-1) + (1.5-1) = 2.5` → 加价后价格 = 原价 × 2.5 |

#### 折扣类系数

| 模式 | 计算方式 | 示例（折扣=0.9，扩展折扣=0.95） |
|------|----------|-------------------------------|
| **乘法模式** | 系数相乘 | `0.9 × 0.95 = 0.855` → 折扣后价格 = 加价后价格 × 0.855 |
| **加法模式** | 系数增量相加 | `1 + (0.9-1) + (0.95-1) = 0.85` → 折扣后价格 = 加价后价格 × 0.85 |

### 20.4 小票显示

**加价合计**显示加价后的总价，**折扣合计**显示减少的金额：

```
制品小计                    ¥1000.00
─────────────────────────────────────
加价合计：3.000×            ¥3000.00
  同人商用：2.000×            +¥1000.00   ← 加法模式显示增减金额
  72H加急：1.500×            +¥500.00
─────────────────────────────────────
折扣合计：0.855×            -¥435.00
  上次合作寄样：0.9×         -¥300.00    ← 加法模式显示增减金额
  扩展折扣：0.95×            -¥135.00
─────────────────────────────────────
报价金额                    ¥2565.00
```

### 20.5 显示规则

| 项目 | 显示格式 | 说明 |
|------|----------|------|
| 加价合计 | `¥加价后总价` | 始终显示 |
| 折扣合计 | `-¥减少金额` | 始终显示 |
| 加法模式系数明细 | `+¥金额` / `-¥金额` | 显示具体增减金额 |
| 乘法模式系数明细 | 只显示名称和系数值 | 乘法模式不显示单独增减金额 |

### 20.6 向后兼容

- 历史数据默认使用乘法模式（兼容旧数据）
- 新数据保存计算模式和系数明细到 quoteData 中

## 二十二、制品顺序调整

### 22.1 功能说明

计算页制品卡片支持上移/下移调整顺序，方便按客户需求或优先级排列制品。

### 22.2 调整方式

- 每个制品卡片右侧有 **⤒**（置顶）、**↑**（上移）和 **↓**（下移）按钮

- 第一个制品无法置顶和上移
- 最后一个制品无法下移
- 调整顺序不影响价格计算结果

- 点击回收站条目的「永久删除」按钮，订单将被彻底删除
- 永久删除后无法恢复，请谨慎操作
