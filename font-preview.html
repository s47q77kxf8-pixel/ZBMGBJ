说明：

• 已排除的字体（已在其他主题使用）：Consolas, Monaco, Source Code Pro, Fira Code, Inconsolata（深色主题已用）

• OCR-A/OCR-B 需要系统安装，Windows 可能默认没有
• Lucida Console、Fixedsys 是 Windows 系统自带字体
• JetBrains Mono 需要单独安装
• 当前经典样式使用：Courier, Letter Gothic, Courier New（打印机风格）
